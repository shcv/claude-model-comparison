Several factors likely contribute to the aggregate gap:

  • User participation: Opus 4.5 starts immediately, creating a live feedback loop. Opus 4.6’s extended planning phase encourages the user to step away, reducing interactive refinement.
  • Task granularity: With Opus 4.5’s faster start, smaller and more frequent requests are natural—creating more sequential edits to the same regions.
